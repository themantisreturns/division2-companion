# Division 2 Companion: Production Roadmap

## Milestone 1 — Catalog foundation (this bundle)
- Versioned generated catalog
- Weekly vendor files used as a safe seed
- Manual source file for categories not present every week
- Catalog loaded independently from vendor recommendations
- Exact-item Expertise UI reads the generated catalog

## Milestone 2 — Catalog completeness
- Import a reviewed community source into `public/catalog/sources/manual.json`
- Add provenance fields to every imported item
- Add a validation report for duplicates and missing categories
- Review names against the in-game Expertise library

## Milestone 3 — Search-first interface
- Global search command
- Item detail drawer
- Filters for category, proficiency, rarity, and vendor availability
- Mobile navigation

## Milestone 4 — Recommendation engine v3
- Exact BUY/SKIP state for all catalogued items
- Roll-threshold upgrades
- Named/exotic collection status
- Route grouping and cost summary

## Milestone 5 — Importers
- Screenshot-assisted Expertise import
- Profile export/import
- Catalog source import format

## Milestone 6 — Production readiness
- Automated tests
- Error reporting
- Privacy page
- Analytics opt-in
- GitHub Pages deployment workflow
