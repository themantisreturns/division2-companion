# v1.9.0 Command Center

The dashboard now turns existing profile data into a focused next-action view.

## Inputs

- Inventory counts and reviewed OCR copies
- Duplicate groups
- Wishlist
- Loot decision history
- Saved build completion
- Personalized weekly vendor recommendations
- Purchased vendor items
- Expertise level

## Dashboard guidance

- **Farm next** prioritizes wishlist items, incomplete saved builds, vendor visits, or scanning new loot.
- **Buy this week** surfaces the highest-scoring unpurchased vendor recommendation.
- **Build focus** points to the least-complete saved build.
- Inventory health and a four-step checklist provide quick maintenance guidance.

The command center is intentionally derived from the same services used elsewhere in the application so its advice stays consistent with Inventory, Builds, and Weekly Vendors.
